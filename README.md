# PulseHR
PulseHR - корпоративный сервис опросов сотрудников для получения обратной связи HR-службе и руководителям.

Пользователь входит в свою учётную запись по номеру телефона, затем необходимо ввести код подтверждения.

Если пользователь не из HR-службы/руководства, на главной странице будут показаны непройденные опросы, а также количество пройденных и непройденных опросов. На панели управления пользователь может перейти в свой профль и выйти из аккаунта.
В профиле можно отредактировать данные о пользователе (имя, почта, подразделение, должность. Телефон менять нельзя), а также настроить, где пользователь будет получать уведомления.

Если же авторизованный пользователь из HR-службы или руководства, помимо опросов на панели управления есть вкладки "Управление", "Аналитика" и "Сотрудники".
Во вкладке "Управление" будут отображаться все созданные опросы. На странице можно создать новый опрос. Каждый созданный опрос можно отредактировать, остановить и показать статистику о нём. Также, отображается статус опроса и его тип (анонимный; критичный).

При создании нового опроса необходимо написать его название, описание (необязательно), дата окончания, примерное время прохождения, роли (кто может пройти опрос) и подразделения. Можно выбрать, будет ли опрос анонимным и можно ли будет отключить уведомления о нём.
Можно создать 5 типов вопросов: одиночный выбор, множественный выбор, шкала оценки (eNPS), текстовый ответ и матрица (таблица). Можно указать, необходимый ли вопрос. Варианты ответов (одиночный, множественный выбор, матрица) указываются в окне каждый на новой строке. Также, можно настроить условия показа вопроса - то есть, можно настроить, показывать ли вопрос, если в каком-то вопросе ранее был указан какой-то определённый ответ.

Во вкладке "Аналитика" нужно указать опрос, по которому HR хочет получить статистику. Можно также выбрать должность и подразделение. Статистика отображается по каждому вопросу опроса в виде диаграмм и процентных соотношений.
Во вкладке "Сотрудники" отображаются все зарегистрированные пользователи. Их можно найти по имени, телефону или подразделению. Также, можно менять их роли и удалять учётные записи.

# Стеки:
- Backend продукта реализован на Flask,
- Frontend - на HTML и CSS,
- Docker-развёртывание.

# Инструкции по запуску:
Необходимо в командной строке перейти к расположению корневой папки приложения, в ней запустить команды:
- docker build -t pulsehr:latest .
- docker save -o pulsehr.tar pulsehr:latest

- docker load -i pulsehr.tar
  
После чего перейти на localhost:5000 (сервер разворачивается на самом компьютере)
